Task-1 :Search “What is JavaScript used for today?” and note 3 real uses.

1. Making websites interactive

JavaScript adds actions and movement to web pages, like:

  • Buttons that do something
  • Sliders, pop-ups, animations
  • Form validation (checking user input)

2. Building full web applications

Modern apps like YouTube, Facebook, Gmaspanil, Instagram all use JavaScript.

Frameworks used:

  • React
  • Angular
  • Vue

3. Backend development (server-side)

With Node.js, JS is used to build:

  • APIs
  • Servers
  • Databases handling

Example: Netflix, PayPal use Node.js.

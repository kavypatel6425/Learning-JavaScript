Task-6 Research difference between HTML, CSS, JS.

1.HTML

HTML = Skeleton of a webpage

  • Stands for HyperText Markup Language
  • Used to create the structure of a webpage
  • Defines headings, paragraphs, images, buttons, forms, etc.

2.CSS

CSS = Design/Look of the webpage

  • Stands for Cascading Style Sheets
  • Used to style the HTML
  • Controls colors, fonts, layout, spacing, animations, etc.

3.JS

JavaScript = Brain/Behavior of the webpage

  • Makes the webpage interactive
  • Used for actions like clicking buttons, showing popups, sliding menus, calculations, etc.

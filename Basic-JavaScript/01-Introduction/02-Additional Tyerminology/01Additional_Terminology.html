Additional Terminology

01.Browser

=>A browser (like Chrome, Firefox, Edge) is a software that displays websites and lets users browse the internet.

02.Console

=>console is a tool used to show messages or results on the screen for developers. We mostly use it to check values and debug the code.

03.Script Tag

=>The script tag is used in HTML to write JavaScript code or link a JavaScript file so the browser can run it.

04.Syntax

=>Syntax in JavaScript means the set of rules that decides how you must write your code so that the computer can understand it.

=>Just like grammar in English, syntax is grammar for JavaScript.

05.Statement

=>A statement in JavaScript is a single instruction that tells the computer to do something.

06.Variable

=>A variable is a container that stores a value.

07.Runtime

=>Runtime means the time when your JavaScript code is actually running in the browser or environment.

=>It is the moment when the code is executed and gives output.

08.Compiler

=>A compiler converts the whole code into machine code before running it.

Interpreter

=>An interpreter reads and runs the code line by line.

09.Error Message

=>An error message is a warning shown by JavaScript when something is wrong in your code, so you can fix it.

10.Developer Tools

=>Developer Tools (DevTools) are built-in tools in the browser that help you check, test, and fix your JavaScript, HTML, and CSS.
